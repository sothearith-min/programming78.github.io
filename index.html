Part 1: Arithmetic Operators

What are Operators?

An operator is a symbol that performs an action on values

Like the buttons on a calculator ‚Äî each does something specific.

Example:

result = 10 + 5   # + is the operator
print(result)     # 15

üí° Think of it as

Operands ‚Üí the values being operated on (10 and 5)
Operator ‚Üí the action (+)

The Seven Arithmetic Operators

Operator Name Example Result
+ Addition 10 + 3 13
- Subtraction 10 - 3 7
* Multiplication 10 * 3 30
/ Division 10 / 3 3.333‚Ä¶
// Integer Division 10 // 3 3
% Modulo (Remainder) 10 % 3 1
** Exponent (Power) 10 ** 3 1000

üí° Division types

/ gives decimals, // gives whole numbers

Addition and Subtraction

Simple and straightforward:

x = 20
y = 8

sum_result = x + y
print(sum_result)        # 28

difference = x - y
print(difference)        # 12

# Can also use with variables
total = 100 - 25 + 10
print(total)             # 85

üî¢ Math works as expected

Left to right for same priority operators.

Multiplication and Division

# Multiplication
price = 12.50
quantity = 4
total_cost = price * quantity
print(total_cost)        # 50.0

# Division (always gives float)
distance = 100
time = 3
speed = distance / time
print(speed)             # 33.333...
print(type(speed))       # <class 'float'>

‚ö†Ô∏è Division always returns float

Even 10 / 2 gives 5.0, not 5

Integer Division (//)

Use // when you want a whole number result:

# Regular division
print(10 / 3)            # 3.3333...

# Integer division (floor division)
print(10 // 3)           # 3

# Useful for counting groups
eggs = 17
dozen = eggs // 12
print(dozen)             # 1 (one complete dozen)

üí° Floor division

It rounds DOWN to the nearest integer.

Modulo Operator (%)

Modulo gives you the remainder after division:

# What's left over?
print(10 % 3)            # 1 (10 = 3*3 + 1)
print(17 % 5)            # 2 (17 = 5*3 + 2)
print(20 % 4)            # 0 (evenly divisible)

# Check even/odd using boolean expressions (no `if` yet)
number = 15
print(number % 2 == 0)   # False -> not even
print(number % 2 != 0)   # True  -> odd
print(f"Is {number} even? {number % 2 == 0}")

üéØ Useful for

  • Checking if divisible (remainder == 0)
  • Finding even/odd numbers
  • Cycling through a range

Exponent Operator (**)

Raise a number to a power:

# Square and cube
base = 2
print(base ** 2)         # 4 (2 squared)
print(base ** 3)         # 8 (2 cubed)

# Powers of 10
print(10 ** 2)           # 100
print(10 ** 3)           # 1000

# Fractional powers (roots)
print(16 ** 0.5)         # 4.0 (square root)
print(27 ** (1/3))       # 3.0 (cube root)

üí° Power notation

2 ** 3 means ‚Äú2 to the power of 3‚Äù = 2 √ó 2 √ó 2

Part 2: Relational Operators

Comparing Values

Relational operators compare two values and return True or False

Like asking a yes/no question about two numbers.

Example:

age = 18
print(age > 17)   # True
print(age < 18)   # False

The Six Relational Operators

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
< Less than 3 < 5 True
> Greater than 5 > 3 True
<= Less than or equal 5 <= 5 True
>= Greater than or equal 5 >= 5 True

‚ö†Ô∏è Comparison vs Assignment

  • == (equals) ‚Äî compares two values
  • = (single) ‚Äî assigns a value to a variable

Equal and Not Equal

# Check if values are the same
x = 10
y = 10
print(x == y)            # True
print(x == 5)            # False

# Check if values are different
print(x != y)            # False
print(x != 5)            # True

# Works with strings too
name = "Sok"
print(name == "Sok")     # True
print(name == "Anna")    # False

üí° Careful with types

"10" == 10 is False (string vs number)

Less Than and Greater Than

score = 75

print(score < 80)        # True
print(score > 80)        # False
print(score > 50)        # True

# Practical example: checking grades using comparisons (no `if` yet)
print("Score >= 90 -> A?", score > 90)
print("Score > 80 and <= 90 -> B?", score > 80 and score <= 90)
print("Score > 50 -> Passing?", score > 50)

üí° Directional thinking

3 < 5 means ‚Äú3 is less than 5‚Äù ‚úÖ

Less Than or Equal / Greater Than or Equal

age = 18

print(age >= 18)         # True (18 is >= 18)
print(age <= 21)         # True
print(age <= 17)         # False

# Practical: age restrictions using comparisons
print("Adult?", age >= 18)
print("Minor?", age < 18)

üîÑ Includes the boundary

<= includes the equal value

Part 3: Logical Operators

Combining Conditions

Logical operators combine conditions to create complex decisions

Ask multiple yes/no questions at once.

Example:

age = 20
has_license = True
# Evaluate the boolean expression directly
print("Can drive?", age >= 18 and has_license)

The Three Logical Operators

Operator Meaning Example Result
and Both must be True True and True True
or At least one True True or False True
not Reverses True/False not False True

üí° Think naturally

‚ÄúAND‚Äù = both true
‚ÄúOR‚Äù = at least one true
‚ÄúNOT‚Äù = opposite

The and Operator

Both conditions must be True:

age = 20
has_money = True

# Both must be True (evaluate directly)
print(age >= 18 and has_money)       # True

# One is False (evaluate directly)
print(age >= 18 and has_money == False)  # False

# Truth table
print(True and True)       # True
print(True and False)      # False
print(False and False)     # False

üéØ Common use

Check multiple conditions that must all be satisfied

The or Operator

At least one condition must be True:

day = "Saturday"
is_holiday = False

# Evaluate the boolean expression directly
print(day == "Saturday" or is_holiday)  # True

# Truth table
print(True or False)       # True
print(False or False)      # False
print(True or True)        # True

üí° Inclusive

Only False when BOTH are False

The not Operator

Reverses True to False and vice versa:

is_raining = False

# Using not (evaluate directly)
print(not is_raining)    # True

# Equivalent ways
print(is_raining == False)  # True

# With variables
is_adult = False
print(not is_adult)     # True

üîÑ Simple negation

not True ‚Üí False
not False ‚Üí True

Combining Multiple Operators

You can chain multiple logical operators:

age = 16
has_money = True
has_permission = True

# All three must be true (evaluate directly)
print(age >= 18 and has_money and has_permission)  # False

# Complex condition (evaluate directly)
print((age >= 16 and has_money) or has_permission)  # True

üí° Use parentheses

Make your logic clear: (a and b) or c

Part 4: Operator Precedence

Precedence Order (High to Low)

From highest (done first) to lowest (done last):

Precedence Operators Examples
1 (Highest) ** Exponent
2 +x, -x, not Unary operators
3 *, /, //, % Multiplication, division
4 +, - Addition, subtraction
5 ==, !=, <, >, <=, >= Relational
6 and Logical AND
7 (Lowest) or Logical OR

üéØ Key Order

  1. ** (exponent) ‚Äî always first
  2. *, /, //, % (multiply/divide)
  3. +, - (add/subtract)
  4. Comparisons
  5. and
  6. or

Example: Arithmetic Precedence

# Example 1: Multiplication before addition
result1 = 2 + 3 * 4
# ‚Üí 2 + 12 = 14 ‚úÖ
print(result1)             # 14

# Example 2: Exponent before multiplication
result2 = 2 * 3 ** 2
# ‚Üí 2 * 9 = 18 ‚úÖ
print(result2)             # 18

# Example 3: Left to right for same priority
result3 = 20 - 5 - 3
# ‚Üí 15 - 3 = 12 ‚úÖ
print(result3)             # 12

‚ö†Ô∏è Same priority = left to right

20 - 5 - 3 is (20 - 5) - 3, not 20 - (5 - 3)

Using Parentheses to Override

Parentheses have the HIGHEST priority:

# Without parentheses: multiply first
result1 = 2 + 3 * 4
print(result1)             # 14 (not 20)

# With parentheses: add first
result2 = (2 + 3) * 4
print(result2)             # 20

# Complex example
result3 = (10 - 2) * (3 + 1)
# ‚Üí 8 * 4 = 32
print(result3)             # 32

üí° When in doubt, use parentheses!

It makes your code clearer and prevents mistakes.

Example: Mixed Operators

# Arithmetic and relational
age = 20
print(age > 18 and age < 65)  # True

# Breaking it down:
# 1. age > 18  ‚Üí True
# 2. age < 65  ‚Üí True
# 3. True and True ‚Üí True

# Another example with arithmetic
score = 85
print(score // 10 == 8)        # True
# 1. 85 // 10 = 8
# 2. 8 == 8 ‚Üí True
